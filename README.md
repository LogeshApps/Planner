# LedgerFlow – Credit & Debit Notes Manager

## 🚀 Quick Start

### Prerequisites
- Install **Node.js** from https://nodejs.org (choose the LTS version)

### Run Locally
Open a terminal / command prompt in this folder and run:

```bash
npm install
npm run dev
```

Then open Chrome and go to: **http://localhost:5173**

### Default Login
- **Username:** admin
- **Password:** admin123

---

## 📦 Build for Production (optional)
```bash
npm run build
```
This creates a `dist/` folder you can upload to any web host.

## 🌐 Deploy to Vercel (free)
1. Push this folder to a GitHub repo
2. Go to https://vercel.com and sign in with GitHub
3. Click "New Project" → select your repo → Deploy
4. You'll get a live URL like `https://ledgerflow.vercel.app`

## 🌐 Deploy to Netlify (free)
1. Run `npm run build`
2. Go to https://netlify.com
3. Drag and drop the `dist/` folder onto the Netlify dashboard
4. Done — live URL instantly!
